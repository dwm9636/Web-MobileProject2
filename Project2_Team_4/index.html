Timeline

1989: Tim Berners-Lee creates the first Web Browser
1990: Creation of Archie, the first search engine; and the launch of Adobe Photoshop, the world's most popular graphic editor
1991: Tim Berners-lee creates the first website using html commands and develops http
1993: Mosaic, one of the first widely available, free web browsers with an image interface; MTV produced the first landing page this year
1994: The first banner ad appeared on hotwire; and Berners-Lee creates the World Wide Web Consortium, responsible for setting standards for the web
1995: Creation of the full-text search engine ALTAVISTA, now Yahoo!; Brenden Eich invents JavaScript; Launch of HTML 2.0
1996: Hakon Wium Lee launches CSS, which is first supported on Microsoft Internet Explorer; first appearance of Macromedia's Flash
1997: Launch of Adobe Dreamweaver, GoLive, and other platforms that allowed for non-technical web development
1998: Launch of Google
2004: Launch of MySpace and Facebook; emergence of social media
2005: Launch of Youtube and Reddit
2007: Launch of the Iphone
2010: Increased need for responsive design by web designers because of the popularity explosion of iphones
2013: Creation of React
2014: W3C finalizes the rules for HTML 5.0
2017: Flash is deemed redundant and Discontinued
2019: Mobile internet usage becomes more prominent than computers
